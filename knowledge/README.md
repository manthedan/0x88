# Tiny Neural Chess Knowledge Graph

This vault is the canonical Markdown knowledge-graph overlay for Tiny Leela.

- Markdown files are the source of truth.
- YAML frontmatter provides typed graph metadata.
- Wikilinks provide human and agent navigation.
- `knowledge/graph/*.json` is generated by `scripts/knowledge_graph_overlay.py`.

Start for agents: `09_agent_context/retrieval_manifest.yaml`.
