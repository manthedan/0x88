# Project brief

Tiny Leela builds small Leela-style chess engines that can run in constrained/browser contexts while still benefiting from modern training, evaluation, and search ideas.

Current portfolio lanes:

- CNN baselines and deployable anchors
- Tactical MoveFormer hybrids
- Tiny BT4 / SquareFormer models
