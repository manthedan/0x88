# Reckless notice

Reckless is an external chess engine by its upstream authors and is licensed AGPL-3.0. This project does not claim ownership of Reckless.

This repository only provides browser/WASI build scripts, local patches, and UI integration for generated Reckless WASM artifacts. If you publish or otherwise distribute those artifacts, publish the corresponding patched Reckless source generated by:

```sh
npm run reckless:build-production
```

That command emits the production scalar/SIMD WASM files and matching `*corresponding-source*.tar.gz` archives under `public/reckless/`.
