# Teacher Label Schema

Real distillation uses JSONL records generated by `scripts/query_lc0_teacher.py`.

Each line:

```json
{
  "id": "opening_start_e4",
  "fen": "rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1",
  "policy": { "e2e4": 0.42, "d2d4": 0.28 },
  "wdl": [0.34, 0.33, 0.33],
  "q": 0.01,
  "teacher": "lc0",
  "nodes": 64
}
```

Rules:

- `policy` keys are UCI moves using `docs/move_encoding.md`.
- `policy` probabilities must sum to 1.
- `wdl` is `[win, draw, loss]` from side-to-move perspective and must sum to 1.
- `q` is side-to-move value in `[-1, 1]`.
- Dataset changes require a new dataset id and usually a new benchmark id.

Bootstrap command sequence:

```bash
python3 scripts/export_seed_fens.py
LC0_BIN=/path/to/lc0 LC0_WEIGHTS=/path/to/weights.pb.gz \
  python3 scripts/query_lc0_teacher.py --positions data/distill_seed_positions.fen --nodes 64 --multipv 4
python3 training/validate_teacher_labels.py data/teacher_labels.jsonl
```
